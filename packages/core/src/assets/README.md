# AdyaUI Logo Assets

This directory contains all logo variations for the AdyaUI component library.

## Available Logo Files

### Main Logo

- **logo.png** - Full logo with hexagon icon and "AdyaUI" text (vertical layout)
  - Use for: Splash screens, about pages, documentation headers
  - Dimensions: 600x600px

### Icon Only

- **logo-icon.png** - Hexagon icon without text
  - Use for: App icons, social media avatars, small spaces
  - Dimensions: 400x400px

### Horizontal Layout

- **logo-horizontal.png** - Icon and text side-by-side
  - Use for: Website headers, navigation bars, email signatures
  - Dimensions: 600x200px (approximate)

### Monochrome Version

- **logo-monochrome.png** - Grayscale version
  - Use for: Print materials, single-color applications
  - Color: Dark gray (#333333)

### White Version

- **logo-white.png** - White version for dark backgrounds
  - Use for: Dark mode interfaces, dark hero sections
  - Color: Pure white (#FFFFFF)

### Favicon

- **favicon.png** - Small icon for browser tabs
  - Use for: Website favicon, browser bookmarks
  - Dimensions: 32x32px (optimized for small display)

## Logo Design

The AdyaUI logo features:

- **Geometric Design**: Hexagon with nested triangles
- **Color Scheme**: Gradient from purple (#8e2de2, #4a00e0) to cyan (#00c6ff, #0072ff)
- **Style**: Modern, technical, clean lines
- **Typography**: Bold sans-serif for "AdyaUI" text

## Usage Guidelines

### Do's ✅

- Maintain clear space around the logo (minimum 20px)
- Use the appropriate version for your background (colored vs white)
- Scale proportionally - never stretch or distort
- Use high-resolution versions for print

### Don'ts ❌

- Don't change the colors or gradient
- Don't rotate or skew the logo
- Don't add effects (shadows, outlines, etc.)
- Don't place on busy backgrounds that reduce visibility

## Integration

The logo is already integrated into:

- `index.html` - Demo page header
- Favicon reference in HTML head

## File Formats

All logos are provided in PNG format with transparent backgrounds (where applicable) for maximum compatibility and quality.
