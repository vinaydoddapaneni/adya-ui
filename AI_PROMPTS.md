## üí° Tips for Better Prompts

1. **Be Specific**: Include exact fields, buttons, and layout details
2. **Mention Styling**: Specify colors, themes, or design styles
3. **Include Interactions**: Describe what happens on click, hover, etc.
4. **Specify Layout**: Mention grid columns, alignment, spacing
5. **Add Context**: Explain the use case (e.g., "for a SaaS app")

### Example of a Great Prompt:

```
Create a modern dashboard for a SaaS analytics app. Include:
- Top navbar with logo, search bar, and user menu
- Left sidebar with navigation icons (Dashboard, Analytics, Reports, Settings)
- Main content area with:
  - Welcome message with user's name
  - 4 metric cards in a row (Users, Revenue, Conversion Rate, Active Sessions)
  - Line chart showing revenue over last 30 days
  - Recent activity table with 5 rows
Use a clean, professional design with blue as primary color and support dark mode.
```
